# Product Inventory Tracker 📦

A Python command-line application for managing product inventory and generating monthly stock statements.

## Features
- Enter product details (name, code, sale price, manufacturing cost)
- Log units manufactured and sold for each month
- View product details and monthly unit breakdown
- Generate a full stock statement with net profit calculation

## How to Run
python main.py

## Menu Options
| Option | Description |
|--------|-------------|
| 1 | Enter units manufactured in a month |
| 2 | Enter units sold in a month |
| 3 | View product details |
| 4 | View units manufactured in a month |
| 5 | View units sold in a month |
| 6 | Generate stock statement |
| 0 | Exit |

## Files
| File | Description |
|------|-------------|
| `main.py` | Entry point and application menu logic |
| `product.py` | Product class with getters, setters, and stock tracking |

## Requirements
Python 3.x — no external libraries needed.
